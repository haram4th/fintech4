CSS Selector Practice

CSS Selector 연습문제 100 — 문제/정답(접기)

조건: HTML 구조/태그 추가는 최소화, 기존 HTML 안에서 속성(id/class/title/href/target 등)만 최소 변경. CSS 변경은 글자색(color), 배경색(background-color), 폰트 크기(font-size), 정렬(text-align)만 사용.

1) 태그/기본 2) id/class 3) 속성 선택자 4) 결합자 5) 가상/구조
실습 팁
  • 정답(예시 CSS)은 “정답 보기”를 펼치면 나옵니다.
  • 문항에 “HTML 최소 수정”이 있으면 기존 태그의 속성만 (예: title 추가, href 값 변경, class 추가) 하세요.
  • 가능하면 기존 CSS를 “덮어쓰기”로 연습하세요(우선순위/특이성도 같이 익혀짐).

정답 CSS(예시)

확인 포인트

Q17. (우선순위) 모든 p는 회색, 그런데 “글자형태 태그” 섹션 안의 p만 파란색으로 바꿔라.

HTML 구조는 바꾸지 말고, 선택자를 더 구체적으로.

정답 보기
정답 CSS(예시) 
p { color: gray; }
/* '글자형태 태그'가 들어있는 두 번째 큰 div를 정확히 집기 어려우면

주의: div p는 모든 div 안의 p에 걸려 범위가 넓어질 수 있어요. (이게 바로 연습 포인트)

포인트: hr 바로 다음 형제 div 안의 h1만 잡는 연습.

Q20. (글자 크기) 모든 h2는 18px, 그런데 제목 성격의 h2만 26px로 만들어라.

.browncolor { color: brown; }
Q24. (복합 class) 특정 p에 class="p1 strong center browncolor"를 넣고, 폰트 크기가 15px로 바뀌는지 확인해라.

정답 보기
HTML(최소 수정 예) 
<p class="p1 strong center browncolor">복합 클래스 테스트</p>
관련 CSS(제공된 것) 
.p1 { font-size: 30px; }
.p1.strong.center.browncolor { font-size: 15px; }
Q25. (복합 class) Q24에서 class 순서를 바꿔도(예: browncolor center strong p1) 같은 결과인지 확인해라.
정답 보기

결과: 동일하게 적용된다. class는 순서가 아니라 “포함 여부”로 매칭된다.
